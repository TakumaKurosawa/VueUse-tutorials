## Prerequisite

- Node.js installed
- npm installed

## Setup

```bash
### Install dependencies
npm install

### Compile and Hot-Reload for Development
npm run dev

### Type-Check, Compile and Minify for Production
npm run build

### Preview Production Build
npm run preview

### Lint with [ESLint](https://eslint.org/)
npm run lint
```
